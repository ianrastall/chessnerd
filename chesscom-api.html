Chess.com API

Complete mirror of Chess.com's Published Data API. Use this as your reference for REST endpoints.

The PubAPI is a read-only REST API that responds with JSON-LD data. The API re-packages public data from the website and makes it available via the PubAPI. "Public Data" is information available to people who are not logged in, such as player data, game data, and club/tournament information. This excludes private information that is restricted to the logged in user, such as game chat and conditional moves.

This is read-only data. You cannot send game-moves or other commands to Chess.com from this system.

To use the PubAPI:

  1. determine the data you want, and compose the URL for it based on the endpoint URL pattern
  2. request that URL in your browser, program, Postman, cURL, or pigeon.
  3. enjoy the JSON :)

This page is the documentation for the PubAPI.

General Use Features and Issues

The format for each endpoint will be different, and described in a release note about that endpoint. They all share the following features and issues.

Cache invalidation: These endpoints refresh at most once every 12 hours

Rate Limiting

Your serial access rate is unlimited. If you always wait to receive the response to your previous request before making your next request, then you should never encounter rate limiting.

However, if you make requests in parallel, you should be prepared to accept a 429 Too Many Requests response.

HTTP Responses

  • 200 = "enjoy your JSON"
  • 301 = URL moved
  • 304 = Not Modified (cached)
  • 404 = Not found
  • 410 = Gone (no data will ever be available here)
  • 429 = Too Many Requests

Endpoints

Player Profile

URL pattern: https://api.chess.com/pub/player/{username}

{
  "url": "URL", 
  "username": "string", 
  "player_id": 41, 
  "title": "string", 
  "status": "string", 
  "name": "string", 
  "avatar": "URL", 
  "location": "string", 
  "country": "URL", 
  "joined": 1178556600, 
  "last_online": 1500661803, 
  "followers": 17,
  "is_streamer": false,
  "twitch_url": "Twitch.tv URL",
  "fide": 0
}

Titled Players

URL pattern: https://api.chess.com/pub/titled/{title-abbrev}

Valid title abbreviations: GM, WGM, IM, WIM, FM, WFM, NM, WNM, CM, WCM.

{
  "players": [
    "magnuscarlsen",
    "hikaru",
    ...
  ]
}

Player Stats

URL pattern: https://api.chess.com/pub/player/{username}/stats

{
  "chess_daily": {
    "last": { "rating": 1642, "date": 1509709165, "rd": 58 },
    "best": { "rating": 2065, "date": 1256228875, "game": "URL" },
    "record": { "win": 177, "loss": 124, "draw": 21 }
  },
  "chess_blitz": { ... },
  "tactics": { ... },
  "puzzle_rush": { ... }
}

Player online status

URL pattern: https://api.chess.com/pub/player/{username}/is-online

{ "online": true }

Player Games

There are multiple endpoints for games.

Current Daily Chess

https://api.chess.com/pub/player/{username}/games

To-Move Daily Chess

https://api.chess.com/pub/player/{username}/games/to-move

List of Archives

https://api.chess.com/pub/player/{username}/games/archives

Monthly Archive

https://api.chess.com/pub/player/{username}/games/{YYYY}/{MM}

PGN Download

https://api.chess.com/pub/player/{username}/games/{YYYY}/{MM}/pgn

Player's Clubs

URL pattern: https://api.chess.com/pub/player/{username}/clubs

{
  "clubs": [
      {  
         "name": "Club Name",
         "url": "https://www.chess.com/club/...",
         "joined": 1234567890
      }
  ]
}

Tournaments

URL pattern: https://api.chess.com/pub/tournament/{url-ID}

{
    "name": "Tournament Name",
    "status": "finished",
    "settings": { ... },
    "players": [ ... ],
    "rounds": [ ... ]
}

Country Profile

URL pattern: https://api.chess.com/pub/country/{iso}

{
  "name": "Italy",
  "code": "IT"
}

Country Players

URL pattern: https://api.chess.com/pub/country/{iso}/players

{ "players": [ "user1", "user2", ... ] }

{
   "title": "Puzzle Title",
   "url": "URL",
   "fen": "FEN",
   "pgn": "PGN",
   "image": "URL"
}

Streamers

URL pattern: https://api.chess.com/pub/streamers

{
   "streamers":[
      {
         "username": "hikaru",
         "twitch_url": "..."
      }
    ]
}

Leaderboards

URL pattern: https://api.chess.com/pub/leaderboards

{
   "daily": [ { "username": "...", "score": 2800, "rank": 1 }, ... ],
   "live_blitz": [ ... ]
}
